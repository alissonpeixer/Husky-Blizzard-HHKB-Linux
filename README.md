# Map Arrow keys for Linux  usen Xmodmap!



<section>

  Usando | W A S D
```
  keycode 105 = Caps_Lock
  keycode 66 = Mode_switch  
  keysym a = a A Left
  keysym s = s S Down
  keysym w = w W Up
  keysym d = d D Right
```  
  Usando | H J K L
```
  keycode 105 = Caps_Lock
  keycode 66 = Mode_switch  
  keysym h = h H Left
  keysym j = j J Down
  keysym k = k K Up
  keysym l = l L Right
```

  1ยบ Creat a file in <b>home/pessoal files</b> for name ```.Xmodmap```.
  
  2ยบ Open terminal paste ```xmodmap .Xmodmap```.
